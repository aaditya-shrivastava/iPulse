**IPulse — IP Intelligence Dashboard**

A full-stack web application that takes any IP address and instantly reveals a complete profile — where it's located, who owns it, what type of connection it is, whether it's a VPN or proxy, and even the live weather at that location. Built with HTML, CSS, JavaScript on the frontend and Node.js + Express on the backend, it consumes multiple free REST APIs to enrich the data and displays everything on a clean, responsive dashboard with an interactive map, dark mode, search history, and a threat-level badge
